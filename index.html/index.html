📌 วิธีการใช้งาน:
1. ตั้งค่าการเชื่อมต่อ Google Sheets ด้านบน
2. เลือกแท็บตามด้านที่ต้องการประเมิน
3. กรอกข้อมูลครัวเรือนและผู้ประเมิน
4. ประเมินตามเกณฑ์ที่กำหนด
5. บันทึกผล - ข้อมูลจะถูกส่งไปยัง Google Sheets อัตโนมัติ
6. ดูสรุปผลใน Dashboard
